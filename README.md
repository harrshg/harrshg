## Featured projects

### DocuChat AI — RAG document & log analyser
> Python · LangChain · Gemini API · ChromaDB · Sentence Transformers · Streamlit · Docker

End-to-end RAG pipeline that lets you chat with PDFs, Jenkins CI/CD logs, and build reports. Upload a log file, ask "why did this build fail?", and get a grounded answer with exact line citations.

- PDF/log ingestion → chunking (500 tokens, 50 overlap) → vector embeddings → semantic retrieval → LLM-generated answer
- Extended to ingest CI/CD output for automated failure triage — directly inspired by my work at Graebert
- Dockerized and deployed on Hugging Face Spaces
